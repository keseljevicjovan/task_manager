# KumaraVel

Минималистички PHP framework инспирисан Laravel-ом.

## Захтијеви

- PHP (8.0+)
- MySQL/MariaDB

# Коришћење

Покретање миграција:
```sh
php artisan migrate
```

Покретање локалног development сервера:
```sh
php artisan serve
```

Све остале команде можете добити преко
```sh
php artisan
```
